The following abbreviations are used in the documentation:

Hazard Event (of earthquake/tsunami/storm surge hazard)

GM

Ground Motion (of earthquake hazard)

IM

Intensity Measure (of hazard event)

SAM

Structural Analysis Model (i.e. finite element model)

SIM

Simulation

UQ

Uncertainty Quantification

RV

Random Variables

QoI

Quantities of Interest
