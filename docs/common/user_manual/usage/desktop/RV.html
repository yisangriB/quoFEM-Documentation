4.3. RV: Random Variables¶

The RV panel allows the user to specify the probabilistic distribution for the random problem at hand. The following probabilistic distributions for the random variables are currently supported:

  1. Normal (Gaussian)

    User provides the mean (\(\mu\)) and standard deviation (\(\sigma\)) of the normal distibution. The density function of the normal distribution, as a function of \(\mu\) and \(\sigma\) is:

    (4.3.1)¶\[f(x) = \frac{1}{\sqrt{2 \pi} \sigma} \exp \left( -{\frac{1}{2} \left( \frac{x - \mu}{\sigma} \right)^2} \right)\]

  2. Lognormal

    User provides the mean (\(\mu\)) and standard deviation (\(\sigma\)) of the lognormal distibution. The density function of the lognormal distribution, as a function of \(\mu\) and \(\sigma\) is:

    (4.3.2)¶\[f(x) = \frac{1}{\sqrt{2 \pi} \zeta x} \exp \left( -{\frac{1}{2} \left( \frac{ln x - \lambda}{\zeta} \right)^2} \right)\]

where \(\zeta^2 = \ln \left( \frac{\sigma^2}{\mu^2} + 1 \right)\) and \(\lambda = \ln(\mu) - \frac{\zeta^2}{2}\)

  1. Beta

    User provides \(\alpha\), \(\beta\), lower bound (\(L_B\)), and an upper bound (\(U_B\)) for the beta distibution. The density function of the normal distribution, as a function of these quantaties is:

    (4.3.3)¶\[f(x) = \frac{\Gamma(\alpha + \beta)}{\Gamma(\alpha)\Gamma(\beta)} \frac{(x - L_B)^{\alpha-1}(U_B-x)^{\beta-1}}{(U_B - L_B)^{\alpha + \beta - 1}}\]

    where \(\Gamma(\alpha)\) is the Gamma function.

  2. Uniform

    User provides the lower bound (\(L_B\)), and an upper bound (\(U_B\)) for the uniform distibution. The density function of the normal distribution, as a function of these quantaties is:

    (4.3.4)¶\[f(x) = \frac{1.0}{(U_B - L_B)}\]

    The mean of the distribution is \(\frac{(U_B + L_B)}{2.0}\)

  3. Weibull

    User provides shape parameter (\(k\)) and scale parameter (\(\lambda\)) for the Weibull distibution. The density function of the Weibull distribution, as a function of these quantaties is:

    (4.3.5)¶\[f(x) = \frac{k}{\lambda}\left(\frac{x}{\lambda}\right)^{k-1} \exp \left( -(x/\lambda)^{k} \right)\]

    where \(k,\lambda > 0\) and \(x >= 0\). For \(x<0\), \(f(x) = 0\).

  4. Gumbel

    User provides \(\alpha\) and \(\beta\) for the Gumbel distibution, where \(\beta\) is known as the location parameter and \(\frac{1}{\alpha}\) the scale parameter. The density function of the Gumbel distribution, as a function of these quantaties is:

    (4.3.6)¶\[f(x) = \alpha e^{-\alpha(x-\beta)} \exp(-e^{-\alpha(x-\beta)})\]

For each random variable, the user must enter a name and select from the pull down menu the distribution associated with the random variable. For the distribution selected, the user must then provide the input arguments, which are as described above. Note

To add a new random variable the user presses the Add button. To remove a random variable, the user must first select it by checking the small circle before the random variable, and then pressing the Remove button.

Warning

Removing a random variable may have unintended consequences and cause the UQ Engine to fail.
