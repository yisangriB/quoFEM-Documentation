Release Notes¶

Version 4.2 (Current)¶

Release date: Feb. 2025

Highlights

  1. Implemented new EDP options for calculating wind loads on components and cladding of a building.

  2. Defined JSON scheme for specifying the geometry of component and cladding elements from basic shapes and transformation.

  3. Added automated post-processing functionality for computing area averaged peak loads.

  4. Added new numerical example for demonstrating the feature for a two story low-rise building.

  5. Minor bug fixes to WE-UQ and backend applications.

Version 4.1¶

Release date: Sept. 2024

Highlights

  1. Implemented new functionality to visualize and export overall/pressure loads on building for performance based wind engineering analysis.

  2. New capability to surrogate structural responses and use the surrogate model in the workflow.

  3. Added numerical example for simulating wind loads on a building with surroundings assuming a idealized topology.

  4. Minor bug fixes to pre-/post-processing of the CFD-based events.

Version 4.0¶

Release date: Aug. 2024

Highlights

  1. Added CFD functionality to simulate wind loads on isolated buildings without running the entire workflow in WE-UQ

  2. Major changes required for interfacing with DesignSafe the newly released Version 3 of Tapis. Updated major version to reflect preferences needed for users to submit jobs to TACC HPC resources.

Version 3.4¶

Release date: May. 2024

Highlights

  1. Simulate wind loading on structures surrounded by an array of buildings. Surrounding buildings heights may be randomized or user-defined.

  2. Minor bug fixes in the tool.

Version 3.3¶

Release date: Apr. 2024

Highlights

  1. Simulate wind loading on structures surrounded by an array of buildings. Surrounding buildings heights may be randomized or user-defined.

  2. A new wind load generator module: wind-tunnel informed stochastic wind pressure generator

  3. Empty domain simulation tool.

Version 3.2¶

Release date: Mar. 2024

Highlights

  1. Implemented a new capability to characterize Atmospheric Boundary Layer (ABL) flows using large-eddy simulation (LES).

  2. Added advanced options for pre-processing, monitoring and postprocessing CFD simulation.

  3. Integrated a new method to Turbulence Inflow Tool (TInF) tool for wind loading simulation.

  4. Supports visualization of the simulation data using Paraview on local machine.

  5. Addressed bugs in the prior release

  6. Created a new example to demonstrate the recently added CFD features.

  7. Option for multi-fidelity sampling, allowing for FEM models of different fidelity

Version 3.1¶

Release date: Oct. 2023

Highlights

  1. Added capability to simulate wind load on building with arbitrary shape

  2. Added options to import and transform the STL geometry of the building.

  3. Fixed bugs in the prior release

  4. Created a new CFD example to demonstrate the recently added features.

Version 3.0.0¶

Release date: July. 2023

Highlights

  1. New CFD-based event for wind load evaluation:

  1. Implemented CFD modeling option for generic isolated rectangular buildings.

  2. Added advanced feature for boundary condition specification including inflow and ground roughness representation.

  3. Supports automated monitoring of integrated and cladding loads on the study building.

  4. Supports different turbulence modeling options including Large-Eddy Simulation (LES), Reynolds-Averaged Navier-Stokes (RANS) and Detached-Eddy Simulation (DES).

  5. Updated solver selection (with PISO, SIMPLE and PIMPLE algorithms) and added new controls for numerical setup.

  6. Included support for different OpenFOAM distributions starting from v7 up to v10.

  1. Improved GUI design

  1. Implemented user-friendly GUI support for creating the CFD model

  2. Added a new 3D model visualization window using Visualization Toolkit (VTK)

  3. Supports reading and writing of the case files to the user’s local machine

  1. Automated mesh generation

  1. Integrated automated mesh generation workflow with several refinement controls.

  2. Added option to generate mesh on the user’s local machine.

  1. Examples

  1. Added a new CFD example to demonstrate the recently added features with uncertainty quantification.

Version 2.3¶

Release date: March. 2022

Highlights

  1. New Digital Wind Tunnel Event

  2. New UQ options for sampling, sensitivity, reliability

  3. Updated FEM analysis options

  4. Updated TInF to avoid negative length scales during OpenFOAM run

  5. Minor bug fixes

Version 2.0¶

Release date: September. 2019

Highlights

  1. Added new UQ features

  2. Added the CWE interface to the tool to aid beginners in performing CFD simulations

  3. Integrated with wind tunnel datasets

  4. Updated example files

Version 1.0 (First Release)¶

Release date: July. 2019

Highlights

  1. High-Rise with Pressure Database (DEDM_HRP)

  2. Stochastic Wind Load Generator

  3. Expert CFD Simulation Option

  4. Selection from Existing Wind Event applications
