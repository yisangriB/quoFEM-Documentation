3.2. Download the Application¶

To download the quoFEM app, first navigate to the quoFEM Download page. As shown in Fig. 3.2.4, from the application page you must click on the link Download app and User Manual which will bring you to another page. This page contains a list of downloadable files and directories. There are at least four files available for download from this page:

  1. The PDF file is the User Manual that you are reading now.

  2. The MOV file is an video that provides an introduction to the usage of the application.

  3. The ZIP file is an archive that contains the application files for a Windows operating system.

  4. The DMG file is an archive that contains the application files for a Mac OS X operating system.

Fig. 3.2.4 quoFEM download page.¶

To download the quoFEM app click on the link for the appropriate file for your operating system. This will bring up a pop-up window. Click on the Download button in bottom right hand corner of this pop-up. Unpackage the ensuing downloaded file and place it in a location on your filesystem.

Note

  1. Windows: On computers running the Windows operating system we suggest placing the directory in the C:\SimCenter folder. Once here, create a shorcut link to the application and move this shortcut link to your Desktop folder.

  2. MacOS: We sugget copying the app to your Desktop.

Now test that the application starts. To do this navigate to the location where you placed the application and open it. You should see the user interface (UI) shown in Fig. 3.2.8 after starting the application. Now quit the application. Additional installation steps, outlined in Section 3.1, are required before computations can be performed.

Fig. 3.2.8 quoFEM application on startup.¶

Note

  1. On a Mac you will need to right click on the .dmg file to open it. The UI will not start correctly while in the DMG file, you need to open the .dmg file and then copy the quoFEM app application to your Documents or Desktop folder. You can then move the .dmg file to the trash or eject it after this has been done.

  2. The SimCenter is not recognized as either a Windows or an Apple vendor. Our applications are not recognized by the operating system as being signed. Consequently, you may receive a warning message when you start the quoFEM app application for the first time. Follow the procedure you used in Section 3.1 to remedy this issue.

The quoFEM app application requires additional software outlined in Section 3.1 to work properly. In Section 3.3, instructions are given on testing the installation.
