Download the Application¶

To download the quoFEM app, first navigate to the quoFEM Download page. As shown in figDownloadQUO_FEM, from the application page you must click on the link Download app and User Manual which will bring you to another page. This page contains a list of downloadable files and directories. There are at least two files available for download from this page:

  1. The ZIP file is an archive that contains the application files for a Windows operating system.

  2. The DMG file is an archive that contains the application files for a Mac OS X operating system.

quoFEM download page.¶

To download the quoFEM app click on the link for the appropriate file for your operating system: file ending with Windows_Download.zip for users on a Windows machine and file ending with Mac_Download.dmg for users on a Mac. This will bring up a pop-up window. Click on the Download button in bottom right hand corner of this pop-up. Unpackage the ensuing downloaded file and place it in a location on your filesystem.

Note

  1. Windows: On computers running the Windows operating system we suggest placing the directory in the C:\SimCenter folder. Once here, create a shorcut link to the application and move this shortcut link to your Desktop folder.

  2. MacOS: We sugget copying the app to your Desktop.

Now test that the application starts. To do this navigate to the location where you placed the application and open it. You should see the user interface shown in Fig. 1.2.11 after starting the application. Now quit the application. Additional installation steps, outlined in Install Python, are required before computations can be performed.

quoFEM application on startup.¶

Note

  1. On a Mac you will need to right click on the .dmg file to open it. The user interface will not start correctly while in the DMG file, you need to open the .dmg file and then copy the quoFEM app application to your Documents or Desktop folder. You can then move the .dmg file to the trash or eject it after this has been done.

  2. The SimCenter is not recognized as either a Windows or an Apple vendor. Our applications are not recognized by the operating system as being signed. Consequently, you may receive a warning message when you start the quoFEM app application for the first time. Follow the procedure you used in lblDownloadOther to remedy this issue.

The quoFEM app application requires an up to date version of Python, that outlined in lblDownloadOther to work properly. In Section 1.3, instructions are given on testing the installation.
